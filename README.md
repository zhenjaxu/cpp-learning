# Cpp-Learning

> C++ 系统学习笔记与代码实践。涵盖现代C++特性（STL、智能指针、并发）、LeetCode算法题解以及游戏开发学习。

## 📁 仓库结构

| 目录 | 内容说明 |
|------|---------|
| [`Concurrency/`](./Concurrency) | C++11/14/17 并发编程：线程、锁、条件变量、基于锁的并发数据结构 |
| [`Leetcode/`](./Leetcode) | LeetCode Hot 100 题解，包含思路注释与复杂度分析 |
| [`Game Programming Patterns`](./Patterns/) | 游戏模式学习：游戏循环、更新模式、组件模式、状态模式 |
| [`Game Programming in C++`](./Programming/) | 游戏开发学习：SDL2 图形库、游戏循环与游戏对象、CMake构建 |

## 🔨 技术栈

- **语言**: C++14
- **构建**: CMake, MinGW
- **图形**: SDL2
- **版本控制**: Git
- **环境**: Windows，Ubuntu虚拟机

## 🛠️ 游戏项目

1. [`tetris-sdl2`](https://github.com/zhenjaxu/tetris-sdl2)：基于SDL2实现的俄罗斯方块，实现软降、硬降、幽灵块等玩法。
